# ⌨️ kanata キーバインド早見表

yamy(`.mayu` / `104on109.mayu`)からの移植。**OS は JIS(109) のまま**使用。

> **前提**
> - 物理キーは日本語(JIS)配列。記号は kanata 内で US 配列に変換して出力。
> - **IME** の ON/OFF は 変換/無変換キー（IME 側で「変換=ON・無変換=OFF」に割当）。
> - **英数→Ctrl** だけは OS のレジストリ(Scancode Map)で実現（kanata 管轄外）。

---

## 修飾・特殊キー（tap / hold）

| 物理キー | タップ | 長押し(hold) |
|---|---|---|
| 英数 (CapsLock) | — | **左Ctrl** （OS層 / Scancode Map） |
| Space | 空白 | **Shift** （SandS） |
| 変換 | 変換 (IME) | **nav レイヤ** |
| 無変換 | 無変換(IMEオフ) + **Esc** | **nav レイヤ** |
| `:` キー | `'` / `"` | **Ctrl** |
| `]` キー | `\` / `\|` | **Alt** |

---

## nav レイヤ（変換 or 無変換 を押しながら）

| キー | 動作 |
|---|---|
| `h` `j` `k` `l` | ← ↓ ↑ → |
| `Shift` + `h/j/k/l` | 範囲選択（Shift+矢印） |
| `o` | ウィンドウ切替（Alt+Tab） |
| `i` | 逆方向（Alt+Shift+Tab） |
| `r` | kanata 設定リロード |

---

## 同時押し・マウス・緊急

| 入力 | 動作 |
|---|---|
| `f` + `j` | **Esc**（50ms以内 / nav中は無効） |
| 右Alt | 左クリック |
| 右Win | 中クリック |
| アプリケーションキー | 右クリック |
| 左Ctrl + Space + Esc | **kanata 強制終了** |

---

## 記号 JIS→US 配列（通常 / Shift）

| 物理キー | 通常 | Shift |
|---|---|---|
| 半角/全角 | `` ` `` | `~` |
| `2` | `2` | `@` |
| `6` | `6` | `^` |
| `7` | `7` | `&` |
| `8` | `8` | `*` |
| `9` | `9` | `(` |
| `0` | `0` | `)` |
| `-` | `-` | `_` |
| `^` キー | `=` | `+` |
| `@` キー | `[` | `{` |
| `[` キー | `]` | `}` |
| `;` キー | `;` | `:` |
| `:` キー | `'` | `"` |
| `]` キー | `\` | `\|` |

> ※ `1` `3` `4` `5` の Shift（`!` `#` `$` `%`）は JIS と US で同じなので変換不要。
> `:` キーと `]` キーは長押しで Ctrl / Alt も兼ねる。
